# 高德地图省会城市天气展示

这个项目使用 React、Tailwind CSS、高德地图 API 和和风天气 API，实现了在地图上标记主要省会城市并显示最近七天的天气信息的功能。

## 功能特点

- 在中国地图上标示主要省会城市。
- 点击城市标记可以查看该城市最近七天的天气预报。
- 支持响应式布局，适应不同屏幕大小和设备。

## 技术栈

- **React**: 前端框架
- **Tailwind CSS**: 现代 CSS 框架，用于快速构建界面
- **高德地图 API**: 提供地图显示和地理编码服务
- **和风天气 API**: 提供天气数据

## 运行项目

- npm start
